This is how we make a word appear bold.

Inside a product description you might see some key features in bold.

By enclosing words in the b element tag, you can make the characters appear bold.

The b element also represents a section of text that would be presented in a visually different way (for example key words in a sentence) although the use of the b element does not imply any additional meaning.

This is how we make a word appear italic

It's a potato Solanum teberosum.

Captain Cook sailed to Australia on the Endeavor.

By enlcosing a word in the i tag, you can make characters appear italic. The i element also represents a section of text that would be said in a different way from surrounding content -- such as technical terms, names of ships, foreign words, thoughts, or other terms that would usually be italicized.
